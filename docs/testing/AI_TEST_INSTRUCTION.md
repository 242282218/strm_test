# AI 测试执行指令

## 🤖 给 AI Agent 的话

你好！我是开发这个功能的 AI，现在请你作为 **QA Agent** 来测试智能重命名的夸克云盘集成功能。

### 📄 测试文档
请仔细阅读：`docs/testing/AI_TEST_GUIDE.md`

### ✅ 你的任务
1. **按照测试用例**（TC1-TC10）**依次执行**
2. **记录每个用例的结果**（通过/失败）
3. **截图或记录关键输出**
4. **生成简洁测试报告**（参考文档中的模板）

### 🔧 你可以使用的工具
- `run_command`: 执行验证命令
- `read_terminal`: 查看后端日志
- `browser_subagent`: 自动化浏览器操作（如果可用）
- `view_file`: 查看配置文件

### ⚠️ 重要规则
1. **按顺序测试**：先TC1，再TC2...不要跳过
2. **一次一个**：完成一个用例再进行下一个
3. **记录失败**：如果某个用例失败，详细记录错误信息
4. **性能测量**：记录每个操作的耗时
5. **报告精简**：最终报告不超过50行

### 📊 报告要求
使用以下格式：

```
=== 测试报告 ===
通过: X/10
失败: [TC编号, 原因]
性能: 符合/不符合基准
阻塞问题: 有/无

详细结果:
- TC1: ✅ PASS
- TC2: ❌ FAIL (原因)
...
```

### 🚀 开始测试
现在请执行：
1. 读取测试文档：`docs/testing/AI_TEST_GUIDE.md`
2. 检查前置条件
3. 开始执行 TC1

**祝测试顺利！如果遇到问题，请标记为 BLOCKED 并说明原因。**
